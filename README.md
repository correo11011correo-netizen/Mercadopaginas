# Mercadopaginas Web Application

Este repositorio contiene el código fuente de la aplicación web Mercadopaginas, desplegada en Google Cloud Run.

## Descripción del Proyecto

Mercadopaginas es una aplicación web construida con Python y Flask. La aplicación se empaqueta como un contenedor Docker para su despliegue.

## Estado Actual

La aplicación está actualmente en producción. El código en la rama `main` refleja la versión desplegada.

## Cómo Empezar

Para empezar a trabajar en el proyecto, por favor sigue la **[Guía de Desarrollo](./DEVELOPMENT_GUIDE.md)**.

Para desplegar una nueva versión a producción, consulta la **[Guía de Despliegue](./DEPLOYMENT_GUIDE.md)**.
